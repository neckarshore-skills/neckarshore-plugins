# Neckarshore AI — Claude Code Plugin Marketplace

The plugin marketplace for [Neckarshore AI](https://neckarshore.ai). Add it once in Claude Code, then install any of our plugins.

## Install

```
/plugin marketplace add neckarshore-skills/neckarshore-plugins
/plugin install obsidian-vault-autopilot@neckarshore-ai
```

## Plugins

| Plugin | What it does |
|--------|--------------|
| [`obsidian-vault-autopilot`](https://github.com/neckarshore-skills/obsidian-vault-autopilot) | AI-powered vault autopilot — sorts, renames, tags, and enriches your Obsidian notes automatically. |
| `generic-tools` | Productivity + meta plugins (PRD/brainstorming, plugin-building). *Coming soon.* |

## About

This repository holds only the marketplace manifest (`.claude-plugin/marketplace.json`). Each plugin lives in its own repository under [neckarshore-skills](https://github.com/neckarshore-skills); versions are sourced from each plugin's own `plugin.json`.
